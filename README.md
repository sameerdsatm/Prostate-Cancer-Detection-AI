# Prostate Cancer Detection Using AI, Multi-parametric MRI and 3D Deep Learning
## Overview
This project focuses on automated prostate cancer detection using multi-parametric MRI (mpMRI) and 3D deep learning techniques. The system performs MRI preprocessing, lesion segmentation, cancer classification, and visualization of results.

## Features
- MRI preprocessing and normalization
- 3D Deep Learning based detection
- Lesion segmentation
- Explainable AI using Grad-CAM
- Flask REST API backend
- React.js frontend interface
